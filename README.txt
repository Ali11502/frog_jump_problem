1. Program is designed in a way that an empty space between frogs is denoted by "x" (x in small form), otherwise it will not run properly.
2. Input should be in string form.
3. On left side it will accept only these characters: 'A','B', or’C’.
4. String length could be either 5 or 7.
User will be prompted to enter starting state to generate graph. Then generate-graph is a function which will run and then adjacency list of states will appear .
Then you will be asked to enter source state (initial state) and goal state. It will use these inputs for traceback function which actually gives output. There are separate trace back functions for bfs and dfs.
